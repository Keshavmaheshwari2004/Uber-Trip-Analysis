# 🚖 Uber Trip Analysis Dashboard (Power BI)
📊 Project Overview

This project analyzes Uber trip data using Power BI to identify booking trends, revenue patterns, trip efficiency, and customer behavior. The dashboard helps businesses make data-driven decisions related to pricing strategy, demand forecasting, and operational efficiency.

# The report is divided into 3 interactive dashboards:

# Overview Analysis
# Time Analysis
# Detailed Data View

Business requirements are based on the problem statement provided in .

# 🎯 Business Objective

The goal of this project is to:

• Understand ride booking trends
• Analyze revenue performance
• Identify peak demand time periods
• Study trip distance and duration patterns
• Identify most popular pickup and drop-off locations
• Support better decision-making for pricing and driver allocation

# 🛠 Tools & Technologies Used

• Power BI
• Excel
• Data Modeling
• DAX (Data Analysis Expressions)
• Data Visualization
• Business Intelligence

# 📂 Dataset Information

The dataset includes Uber trip details such as:

• Trip ID
• Pickup Date & Time
• Vehicle Type
• Payment Type
• Number of Passengers
• Trip Distance
• Booking Value
• Pickup Location
• Drop-off Location

# 📋 Dashboard 3: Detailed Data View
Features

• Complete Trip Data Table
• Drill-through functionality from charts
• Filter data by City, Date, Vehicle Type
• View full dataset option

This page helps users explore granular level trip data.

# 📈 Key Insights from Analysis

• Most bookings come from short-distance trips
• Certain locations have higher ride demand
• Weekend bookings are higher than weekdays
• UberX and Uber Comfort are most frequently used vehicles
• Booking value remains consistent around average price
